# Registre des prompts (PRM)

> **Rôle unique :** inventaire et structure des prompts de production. Un prompt ne contient jamais un fait de la méthode en dur : il est assemblé depuis le référentiel (voir Pipeline.md).

## Modèle de fiche prompt

| Champ | Contenu |
|---|---|
| ID | PRM-000 |
| Rôle | _À compléter._ |
| Entrées | Variables consommées (voir Variables.md) |
| Sorties attendues | _À compléter._ |
| Contraintes de rédaction | Renvoi P-01…P-05 + règles de ton |
| Version | v5.0 |
| Jeu de test | _À compléter._ |
