# Référentiel des recommandations (REC)

> **Rôle unique :** registre des actions recommandées. Une recommandation = une action, son effort, son impact, et les contrôles qu'elle corrige. C'est ici que vivent « Votre première action » et « Résultat attendu ».

## Modèle de fiche REC

```yaml
id: REC-000
corrige: [CTL-000]
action: ""                # formulation client
resultat_attendu: ""
effort_minutes: 0
impact: eleve             # tres_eleve | eleve | modere
inclus_pack: true         # livrable par le Pack Visibilité
prerequis: []             # REC-…
statut: brouillon
depuis_version: v5
```
